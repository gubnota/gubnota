Сегодня сравнил между собой скринкаст-видео с экрана в формате gif и html5 видео. html5 видео выиграл, зато gif можно встраивать в микроблоггинг китайский, и прочее.

Программы создающие gif анимацию: GifGrabber и VideoGIF Free. видео создавалось встроенными средствами захвата экрана (QuickTime) и ffmpeg кодировщиком.

# Кодирование видео
ffmpeg -i 201502111820.mov \
-acodec libvorbis -ac 2 -ab 0k -ar 33600 -t 15 \
-b:v 250k -s 684x608 201502111820.webm

ffmpeg -i 201502111820.mov \
-ac 2 -ab 0k -ar 44100 -strict -2 -t 15 \
-b:v 250k -s 684x608 201502111820.mp4

ffmpeg -i 201502111820.mov \
-ac 2 -ab 0k -ar 44100 -t 15 \
-vcodec h264 -quality good \
-b:v 250k -s 684x608 201502111820.flv

ffmpeg -i 201502111826.gif \
-acodec libvorbis -ac 2 -ab 0k -ar 33600 -t 10 \
-b:v 220k -s 480x300 201502111826.webm

ffmpeg -i 201502111826.gif \
-ac 2 -ab 0k -ar 44100 -strict -2 -t 10 \
-b:v 190k -s 480x300 201502111826.mp4

ffmpeg -i 201502111826.gif \
-ac 2 -ab 0k -ar 44100 -t 10 \
-vcodec h264 -quality good \
-b:v 190k -s 480x300 201502111826.flv
